Перелік засобів та медикаментів шкільної аптечки лабораторії хім. кабінету

 

1.     Бинт стерильний (1 упаковка).

2.    Бинт нестерильний (1 упаковка).

3.    Серветки стерильні (1 упаковка).

4.    Вата гігроскопічна стерильна в тампонах (50 г).

Зберігати в стерильній прозорій склянці з притертою пробкою.

5.    Пінцет для накладання ватних тампонів на рану.

6.    Клей БФ-6 для обробки мікротравм (1 флакон, 25-50 мл).

7.    Йоду настоянка для обробки шкіри біля рани (в ампулах або в темному флаконі, 25-50мл).

8.    Пероксид водню (50мл).

9.    Активоване вугілля.

10. Аміачний розчин 10%. Використовується при втраті свідомості та при отруєнні парами брому.

11. Альбуцид (сульфацил натрію) 30% (10-20мл).  Капати в очі після промивання по 2-3 каплі. Зберігати при кімнатній температурі не більше 3-х тижнів.

12. Спирт етиловий (30-50мл), для обробки опіків.

13. Гліцерін 20-30мл для зникнення больових відчуттів після опіку.

14. Водяний розчин гідрокарбонату натрію 2% для обробки шкіри після опіку кислотою (200-250мл).

15. Водяний  розчин борної кислоти 2% для обробки шкіри після  потрапляння лугу.

16. Піпетки 3 шт. для огних крапель.
